# PANSOPS Calculator Suite

**PANSOPS Calculator** is an open-source web application providing a suite of PANS-OPS calculation tools that normally require spreadsheets. It runs entirely in the browser with no installation required.

**[→ Open the live app](https://flyght7.com/webapp/Main.html)** &nbsp;|&nbsp; **[GitHub](https://github.com/FLYGHT7/pansops-calculator)**

## Calculators

| Calculator                                                              | Section    | Description                                    |
| ----------------------------------------------------------------------- | ---------- | ---------------------------------------------- |
| [ISA Deviation](../calculators/general/isa-deviation)                   | General    | ISA temperature deviation and T_REF            |
| [True Airspeed (TAS)](../calculators/general/tas)                       | General    | CAS/IAS → TAS conversion                       |
| [Rate & Radius of Turn](../calculators/general/rate-turn)               | General    | Turn parameters from TAS and bank angle        |
| [DME Tolerance](../calculators/general/dme-tolerance)                   | General    | Fix tolerance for DME-defined positions        |
| [Wind Spiral](../calculators/general/wind-spiral)                       | General    | Wind spiral displacement for procedure turns   |
| [Bearings & Angles](../calculators/general/bearings-angles)             | General    | Magnetic/true bearing conversions              |
| [OMNI SID PDG](../calculators/departures/omni-sid)                      | Departures | Minimum climb gradient for OMNI SIDs           |
| [Profile Estimator](../calculators/approach/profile-estimator)          | Approach   | Approach vertical profile heights              |
| [NPA SOC Calculation](../calculators/approach/npa-soc)                  | Approach   | Start of Climb for NPA missed approach         |
| [VSS/OCS Parameters](../calculators/approach/vss-ocs)                   | Approach   | Visual Segment and Obstacle Clearance Surfaces |
| [Circling Protection Area](../calculators/approach/circling-protection) | Approach   | Circling area by aircraft category             |
| [ILS Height Calculations](../ils/height-calculations)                   | ILS        | Glide path heights at distance fixes           |
| [ILS Distance Calculations](../ils/distance-calculations)               | ILS        | Distances for given glide path heights         |

## Key Features

- **No installation** — runs in any modern browser
- **Dark mode** with automatic OS preference detection
- **Bilingual** — English and Spanish
- **Copy Results** — export as HTML table or tab-separated text
- **Responsive** — desktop, tablet, and mobile
