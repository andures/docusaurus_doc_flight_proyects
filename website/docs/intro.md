# FLYGHT7 Documentation

**FLYGHT7** is an open-source organization developing aviation software tools focused on ICAO PANS-OPS procedure design, aeronautical charting, obstacle analysis, and compliance verification.

All projects are released under the **GPL-2.0 license** and hosted at [github.com/FLYGHT7](https://github.com/FLYGHT7).

---

## Projects

### PANSOPS Calculator

**[→ Documentation](./pansops-calculator/intro)** &nbsp;|&nbsp; [GitHub](https://github.com/FLYGHT7/pansops-calculator) &nbsp;|&nbsp; [Live App](https://flyght7.com/webapp/Main.html)

A browser-based suite of PANS-OPS calculation tools — ISA deviation, TAS, turn parameters, DME tolerance, wind spiral, ILS profiles, approach minima, and more. Requires no installation.

- **Language:** HTML / JavaScript / CSS
- **Status:** Active

---

### qpansopy

**[→ Documentation](./projects/qpansopy)** &nbsp;|&nbsp; [GitHub](https://github.com/FLYGHT7/qpansopy)

An open-source PANS-OPS software implementation built as a **QGIS plugin** (Python). Automates construction of obstacle assessment areas and protection areas directly inside QGIS.

- **Language:** Python
- **Status:** Active

---

### qAeroChart

**[→ Documentation](./projects/qaero-chart)** &nbsp;|&nbsp; [GitHub](https://github.com/FLYGHT7/qAeroChart)

An open-source aeronautical charting solution built on **QGIS**, designed to produce instrument approach charts and aerodrome charts in compliance with ICAO SARPs (Annex 4).

- **Language:** Python
- **Status:** Active

---

### qOLS

**[→ Documentation](./projects/qols)** &nbsp;|&nbsp; [GitHub](https://github.com/FLYGHT7/qOLS)

A QGIS plugin for generating **Obstacle Limitation Surfaces (OLS)** in accordance with ICAO Annex 14 Volume I and Volume II (aerodromes and heliports).

- **Language:** Python
- **Status:** Active

---

### qBRA

**[→ Documentation](./projects/qbra)** &nbsp;|&nbsp; [GitHub](https://github.com/FLYGHT7/qBRA)

A QGIS-based tool for aviation analysis. See the project repository for details.

- **Language:** Python
- **Status:** Active

---

### tofpa

**[→ Documentation](./projects/tofpa)** &nbsp;|&nbsp; [GitHub](https://github.com/FLYGHT7/tofpa)

A **Take-Off Flight Path Analysis (TOFPA)** tool for aviation, implementing ICAO obstacle clearance criteria for the take-off flight path.

- **Language:** QML
- **Status:** Active

---

### Checklists

**[→ Documentation](./projects/checklists)** &nbsp;|&nbsp; [GitHub](https://github.com/FLYGHT7/checklists)

A series of compliance checklists for reviewing **aeronautical charts and procedure design** against ICAO standards.

- **Language:** HTML
- **Status:** Active
