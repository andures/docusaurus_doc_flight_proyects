# qBRA

**qBRA** is a QGIS plugin developed by FLYGHT7 for aviation procedure design and airspace analysis.

**[GitHub →](https://github.com/FLYGHT7/qBRA)**

## Overview

|              |             |
| ------------ | ----------- |
| **Language** | Python      |
| **Platform** | QGIS Plugin |
| **License**  | GPL-2.0     |

## What It Does

qBRA is part of the FLYGHT7 suite of QGIS-based aviation tools. It integrates with QGIS's geospatial environment to assist with specific aviation calculation and visualization tasks.

:::info Documentation In Progress
Detailed documentation for qBRA is under development. Visit the [GitHub repository](https://github.com/FLYGHT7/qBRA) for the latest source code, examples, and issue tracking.
:::

## Prerequisites

- **QGIS 3.x** (LTR recommended)
- Python 3 (bundled with QGIS)

## Installation

1. Clone or download from [GitHub](https://github.com/FLYGHT7/qBRA)
2. Place plugin folder in:  
   `~/.local/share/QGIS/QGIS3/profiles/default/python/plugins/`
3. Enable via **QGIS → Plugins → Manage and Install Plugins**
